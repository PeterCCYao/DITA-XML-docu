How to add numbers

This section demonstrates how to use the addition feature of The Smart Calculator app.

To successfully add numbers, you must know what addition is.

  1. Enter the first number.
  2. Select the plus (+) button.
  3. Enter the second number.
  4. Select the equal (=) button.

The result of addition is displayed.

If you want to add 6 and 4, the equation will look like: 6+4=10. You enter 6, select plus, enter 4, and then select equal. The result 10 appears on the screen.
