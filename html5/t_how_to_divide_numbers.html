How to divide numbers

This section demonstrates how to use the division feature of The Smart Calculator app.

To successfully divide numbers, you need to know what division is.

  1. Enter the first number.
  2. Select the division symbol (รท).
  3. Enter the second number.
  4. Select the equal (=) button.

The result of the division is displayed.

If you want to divide 20 by 4, the equation will look like: 20รท4=5. You enter 20, select division symbol, enter 4, and then select equal. The result 5 appears on the screen.
